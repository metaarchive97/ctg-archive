# YouTube Video Metadata Archive

## Overview

This repository documents metadata of deleted YouTube videos for archival and research purposes.

The archive contains publicly visible information that was captured prior to deletion, including:

- Video titles  
- Thumbnails  
- Dates of publication  

No video files are hosted or redistributed.

---

## Purpose

The purpose of this archive is documentation.

Online content can be deleted, edited, or made unavailable over time. This repo preserves a record of publicly available metadata for historical reference, transparency, and research.

This archive does not attempt to replicate or redistribute the original video content.

---

## Scope & Limitations

Included:
- Title as displayed in search results
- Thumbnail image as displayed in search results
- Publication date (when available)

Not included:
- Video files
- Audio content
- Private or restricted material
- Download links
- Monetization

This repository contains metadata only.

---

## Data Source

The entries were compiled from publicly accessible search engine results and publicly visible YouTube metadata prior to deletion.

No access to private accounts, private videos, or restricted systems was used.
